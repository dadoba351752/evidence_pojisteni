O aplikaci

Tato webová aplikace slouží k evidenci pojištěnců a jejich pojištění. Umožňuje přehledně spravovat seznam klientů, zobrazovat jejich detailní informace a přidávat, upravovat nebo mazat jednotlivá pojištění, která mají sjednaná. Aplikace také obsahuje validaci formulářů a zobrazení chybových nebo informačních hlášek.

Projekt jsem vytvořil po absolvování rekvalifikačního kurzu programování u ITnetwork.cz, jako způsob, jak si upevnit a rozšířit nabyté znalosti. Cílem aplikace bylo propojit teorii s praxí a zlepšit se v návrhu i realizaci jednoduché webové aplikace v Javě.

Použité technologie:

  • Java
  • Spring Boot – backendový framework pro REST a MVC
  • Thymeleaf – šablonovací systém pro vykreslování HTML
  • Spring MVC – řízení toku aplikace a zpracování HTTP požadavků
  • Spring Validation – validace formulářů a vstupních dat
  • Bootstrap – responzivní design a stylování rozhraní
  • HTML/CSS – základní struktura a vzhled stránky
  • Maven – správa závislostí a buildování projektu
